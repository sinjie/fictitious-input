# fictitious-input

手机端虚拟数字键盘，模拟软键盘输入框。

可自定义光标颜色，默认值等等

用法：

```
var input = new FictitiousNumber(ele, [option]) 
// ele: 作为输入框的元素（必传），
// option: 自定义配置（选传）
```

取值：

```
input.value
```

有效文件：

- ```
  fictitious.css
  ```

- ```
  fictitious.js
  ```

直接引入即可，可参考demo.html
